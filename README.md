# Introduction

Project in which a REST API is created to manage cooking recipes in the [Elasticsearch](https://www.elastic.co/es/what-is/elasticsearch) database in order to be able to:

- Create a new recipe
- Retrieve a recipe
- Modify a recipe
- Search a recipe by its name
